# 🛡️ Secure Ticket Management System (STMS)

A modern enterprise-level Secure Ticket Management System developed using **Flask, MySQL, HTML, CSS, JavaScript, and Chart.js**.

This project was developed as an MCA Major Project to simulate a real-world enterprise IT support and ticket handling platform.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/secure-ticket-management-system.git
```

---

## 2️⃣ Open Project Folder

```bash
cd secure-ticket-management-system
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Configure MySQL Database

Update MySQL credentials inside:

```python
app.py
```

```python
app.config['MYSQL_HOST'] = 'localhost'
app.config['MYSQL_USER'] = 'root'
app.config['MYSQL_PASSWORD'] = 'your_password'
app.config['MYSQL_DB'] = 'stms_db'
```

---

## 7️⃣ Run Application

```bash
python app.py
```
